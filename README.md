## 🎓 Learning Journey
This project is my first step into professional web development. It was built as a practical application of the **HTML & CSS** by **Elzero Web School (Osama Elzero)**. 

Through this project, I practiced:

- **CSS Layouts:** Mastering Flexbox and CSS Grid for structured designs.
- **Responsive Design:** Implementing layouts that work on all devices without using frameworks like Bootstrap.
